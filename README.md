📊 Data Jobs Dashboard | Power BI
Overview

An interactive Power BI dashboard analyzing nearly 500,000 global data job postings to uncover salary trends, hiring demand, job distribution, and market insights across different data careers.

The dashboard enables users to explore:

Salary benchmarks
Hiring trends over time
Job popularity
Global hiring distribution
Platform comparisons
Detailed drill-through analysis for every job title

Project Objective

The objective of this project was to build an interactive analytics dashboard that allows users to answer questions such as:

Which data jobs pay the most?
How have job postings changed over time?
Which job boards post the most openings?
Where are data jobs located globally?
What benefits are commonly offered?
Which job titles have the highest salaries?

Business Insights

Some insights discovered from the dashboard include:

Data Engineering roles command the highest median salaries.
Full-time positions dominate the data job market.
LinkedIn is the primary hiring platform.
Only a small percentage of postings explicitly mention no degree requirements.
Remote opportunities represent a relatively small portion of total listings.
Hiring demand fluctuates throughout the year, indicating seasonal recruitment patterns.
